- run				sh run.sh -n nameoffolder
	with nameoffolder the name of the folder to be created
- the option -s causes the analyses only to be run with parameters df=5, alpha=1.0 (hardcoded in run.sh)
- createsd is a folder [nameofexperiment] with an .rdata-file containing all the results
- the file can be analyzed and plots can be created using 3. Max-stable/R/plots.R


- to run only some parts: use sh run_[...].sh -n [nameofexperiment]
- the folder [nameofexperiment] must already exist! it must contain the folders "res" and "logs"
- "res" must contain "simulation.rds" if run_fromlocest is used, and additional files if run_frominclusters is used


- to run with other values for locest_dist and smoothing_distance than specified in parameters.R, use run_fromlocest_override.sh:
	sh run_fromlocest_override.sh nameoffolder locest_dist smoothing_distance  (always runs as if -s were set)
